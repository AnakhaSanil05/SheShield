## Project Description

Smart SOS & Safe Walk is a **web-based women’s safety system**. It provides a **one-tap emergency alert** with live location sharing to trusted contacts, and a **Safe Walk timer** that automatically triggers an SOS if the user does not confirm safety. It is **simple, fast, and mobile-friendly** for real-time protection.

---

## Problem Statement

- Women feel unsafe while traveling alone.  
- Delays in notifying family/friends during emergencies.  
- Lack of simple monitoring solutions for solo walks.  

---

## Solution

- One-tap SOS system.  
- Sends live location to trusted contacts via WhatsApp.  
- Timer-based Safe Walk mode that triggers automatic alerts.

### System Workflow

**Workflow:**  
1. Add trusted contacts.  
2. Start Safe Walk timer if walking alone.  
3. Press SOS in case of emergency or let the timer auto-trigger it.  
4. SOS message sent via WhatsApp with live location to first trusted contact.  

---

## How It Works

### SOS Button
- Sends an emergency message with **current location and timestamp** to the first trusted contact via WhatsApp.  

### Trusted Contacts
- Stores contacts in **localStorage**.  
- Add or remove contacts dynamically.  

### Safe Walk
- Countdown timer starts for solo walks.  
- If user doesn’t confirm safety, **auto SOS** triggers.  
- User can mark “Reached Safely” to stop the timer.
